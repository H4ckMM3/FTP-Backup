FTP Backup для Sublime Text
===========================

Благодарю за установку плагина FTP Backup!

Этот плагин позволяет создавать автоматические бэкапы файлов при их сохранении,
отслеживать версии файлов и многое другое.

## Использование

- Для сохранения файла с бэкапом используйте: Ctrl+Shift+R
- Для вызова мини-панели используйте: Alt+B

## Настройка

Первая настройка: Tools → FTP Backup → Browse Backup Folder
Выберите папку для хранения ваших бэкапов.
